配置

通过jQuery插件方式传入参数或使用data-*配置参数方式。

属性 默认值 描述
item li 切换器子项选择器
active active 选中激活的样式名
except false 是否多选 
                                
                                    <ul class="ebanks"  data-toggle="switcher" data-item=".item" data-active="current"  data-except="true">

事件监听

通过监听元素的select.ui.switcher事件 

   $('#j-switcher-1').on('select.ui.switcher', function(){
                                alert(this.id)
                                })

jQuery插件绑定

我们使用$(element).switcher(options)来绑定切换功能。

$.fn.accordion = function(option){
        var defaults = {
            item: 'li',
            target: '>div',
            active: 'active',
            except: false
        };

        option = $.extend({}, defaults, option);

        // 直接调用
        $(this).switcher(option);

        // 事件监听
        return $(this).each(function(){
            var $items = $(this).find(option.item);
            $(this).on('select.ui.switcher', function(e){
                var $this = $(e.relatedTarget), has = $this.hasClass(option.active), $actived = $items.find(option.target), $target = $this.find(option.target);
                !!!(option.except) && $actived.slideUp();
                $target.stop()[!has ? 'slideUp': 'slideDown']();
            });
        });
    };
